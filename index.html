JavaDoc API Documentation
These instructions assume you are familar with Android Studio, git, and basic Android development. The latest Android build tools are strongly recommended.

Installation

The easiest way to get up and running is to add the AAR file to your Android Studio projects 'libs' folder. To do this:
  1. Clone the Ice GL repo using git clone https://github.com/Terence-D/IceGL.git
  2. There is an AAR compiled for debug already in the repo under /app/build/outputs/aar, although I recommend recompiling yourself
  3. You can modify the controls.png drawable to improve the styling of buttons, but please ensure the new png file matches the format and sizing exactly.

Setting Up Your First Project

  1. Create a new Android Studio project, make sure you target API 15 or higher.
  2. Choose to add a blank activity, accept other options as required
  3. You will need to supply your own TTF Font file for text - many free ones are available online, but please check that the license matches your needs!
  4. The AAR you built in the Installation section above can now be copied into your Android app lib folder
  5. To have it picked up by your application, add the following to your apps build.gradle, modify the app-debug name as required. the repositories section goes under the android header: 
    repositories {
    flatDir {
        dirs 'libs'
    }
}

dependencies {
    compile(name:'app-debug', ext:'aar')
}
  6. You can now open up the Activity you will be using, and modify it to inherit from GLActivity. You also need to add a line in onCreate() to change to the first GLScreen you want the user to see in your application. An example is below: 
    public class MainActivity extends GLActivity
{
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        changeScreen (new WelcomeScreen(this));
    }
}
  7. The last step is to create a new class, this time inherited from GLScreen. A very basic example that will show text on the screen is below: 
    public class WelcomeScreen extends GLScreen
{
    public WelcomeScreen(final GLActivity activity)
    {
        super(activity);

        Label title = new Label(this);
        title.setBottom(11);
        title.setLeft(0);
        title.setWidth(ScreenConfiguration.getHorizontalTiles());
        title.setTextJustification(Label.CENTER_JUSTIFY);
        title.setFontSize(64);
        title.setColor(GLColor.Blue);
        title.setText("Hello World");
    }
}
  8. Congratulations, you have the beginnings of a functional GUI built with OpenGL for your new game or application!
